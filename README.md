# 🎣 Locos Por La Pesca - E-Commerce

Proyecto de tienda online desarrollado como práctica académica y portfolio personal.  
La aplicación permite gestionar productos de pesca, visualizar catálogos y simular una experiencia de compra online.

---

## 🚀 Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript
- Bootstrap
- LocalStorage
- Git & GitHub

---

## 📌 Funcionalidades principales

✅ Visualización dinámica de productos  
✅ Carrito de compras  
✅ Agregado y eliminación de productos  
✅ Persistencia de datos con LocalStorage  
✅ Diseño responsive  
✅ Navegación intuitiva  
✅ Simulación de e-commerce real
